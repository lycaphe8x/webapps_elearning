Chào mừng bạn đến với Sakai

Chào mừng bạn đến Sakai Demo. Cấu hình Demo được tạo ra như là một cách để bạn có được một trường hợp phát triển của Sakai, nhanh và dễ dàng hơn. Để biết thêm thông tin cài đặt một hướng giải quyết Sakai thích hợp mà sản phẩm cần, làm ơn đọc file readme, và thăm trang web của Sakai tại sakaiproject.org, và trang dành cho người phát triển Sakai tại collab.sakaiproject.org.

Thông tin hiển thị ở đây có thể được sửa đổi bởi Administrator Sakai bằng cách chỉnh sửa file sakai.properties giá trị cấu hình "server.info.url" để trỏ đến file html mong muốn.
